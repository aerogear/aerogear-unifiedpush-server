UnifiedPush Server

A server written in Java which allows sending push notifications to different mobile operating systems and web applications

Android Client SDK

A Java library that allows the integration of the UnifiedPush Server into your native Android application

Swift Client SDK

A Swift library that allows the integration of the UnifiedPush Server into your native iOS application

JS Client SDK

A JavaScript library that allows the integration of the UnifiedPush Server into your JavaScript application (WebApplication, Cordova/Ionic)

Cordova Client SDK

A Cordova plugin that connects our JavaScript library to your multi-platform Cordova application

Java Sender API

A Java library that allows sending push notification from your backend services

Node.js Sender API

A Node.js package that allows sending push notification from your backend services

REST Sender API

A set of RESTful endpoints that allow easily sending push notifications from anywhere
